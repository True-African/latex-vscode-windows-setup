# LaTeX Development Environment Setup for VS Code on Windows

A complete, step-by-step guide to setting up a professional LaTeX writing environment using Visual Studio Code on Windows 10/11.

## Features

- Real-time PDF preview as you type
- Syntax highlighting and auto-completion
- One-click compilation
- Bibliography management support
- Figure and table assistance
- Works with any LaTeX project

## Quick Start

1. Follow the [Setup Guide](setup-guide.md)
2. Use the [templates](templates/) to start new documents
3. Check [troubleshooting](troubleshooting.md) if you encounter issues

## Requirements

- Windows 10 or 11 (64-bit)
- Administrator access for initial setup

## What's Included

- Complete installation instructions
- Troubleshooting common issues
- Ready-to-use LaTeX templates
- VS Code configuration files
- PowerShell helper scripts
